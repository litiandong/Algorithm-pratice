## 最短路径算法

  ***1、***<br>
  **RELAX**API<br>
  对于最短路径算法，需要**松弛**技术。对于每个结点v来说，维持一个属性v.d<br>
  用来记录从源结点s到结点v的最短路径的上界。称v.d为s到v的**最短路径的估计**<br>
  同时算法也要维护**v.π**属性，表示源结点s到结点v的最短路径上，结点v的前驱。<br>
  算法运行最初，v.d的值统一为** ∞ **，v.π的值统一为**NULL**。<br>
